## Margins

> [!IMPORTANT]
> The inside margin (gutter) increases with page count to prevent content from being hidden in the binding.

### Margin Requirements by Page Count

| Page Count | Inside (Gutter) Margin | Outside Margin (No Bleed) | Outside Margin (With Bleed) |
|------------|------------------------|---------------------------|------------------------------|
| 24 - 150 pages | **0.375" (9.6 mm)** | ≥ 0.25" (6.4 mm) | ≥ 0.375" (9.6 mm) |
| 151 - 300 pages | **0.5" (12.7 mm)** | ≥ 0.25" (6.4 mm) | ≥ 0.375" (9.6 mm) |
| 301 - 500 pages | **0.625" (15.9 mm)** | ≥ 0.25" (6.4 mm) | ≥ 0.375" (9.6 mm) |
| 501 - 700 pages | **0.75" (19.1 mm)** | ≥ 0.25" (6.4 mm) | ≥ 0.375" (9.6 mm) |
| 701 - 828 pages | **0.875" (22.3 mm)** | ≥ 0.25" (6.4 mm) | ≥ 0.375" (9.6 mm) |

### Safe Area Calculation for Coloring Books (24-150 pages)

For a typical coloring book with **no bleed**:
- **Outside margins**: 0.25" (6.4 mm) minimum on top, bottom, and outer edge
- **Inside (gutter) margin**: 0.375" (9.6 mm) minimum near the spine

#### Example: 8.5" x 11" Portrait (No Bleed)
```
Page Size:     8.5" × 11" (215.9 × 279.4 mm)
Safe Area:     7.875" × 10.5" (200 × 266.7 mm)

Margins:
  Top:         0.25" (6.4 mm)
  Bottom:      0.25" (6.4 mm)
  Outside:     0.25" (6.4 mm)
  Inside:      0.375" (9.6 mm)
```

#### Example: 8.5" x 8.5" Square (No Bleed)
```
Page Size:     8.5" × 8.5" (215.9 × 215.9 mm)
Safe Area:     7.875" × 8.0" (200 × 203.2 mm)

Margins:
  Top:         0.25" (6.4 mm)
  Bottom:      0.25" (6.4 mm)
  Outside:     0.25" (6.4 mm)
  Inside:      0.375" (9.6 mm)
```

## Bleed (Optional)

> [!NOTE]  
> **Bleed is optional for coloring books.** Most coloring pages don't extend to the edge, so bleed is typically NOT needed. Use bleed only if your design intentionally extends to the page edge.

### Bleed Formula

```
WITH BLEED:
  Page Width  = Trim Width + 0.125" (3.2 mm)
  Page Height = Trim Height + 0.25" (6.4 mm)  [0.125" top + 0.125" bottom]
```

### Page Sizes With Bleed (If Needed)

| Trim Size (No Bleed) | Page Size (With Bleed) |
|---------------------|------------------------|
| 8.5" x 8.5" | 8.625" x 8.75" (21.91 x 22.23 cm) |
| 8.5" x 11" | 8.625" x 11.25" (21.89 x 28.54 cm) |
| 8.25" x 8.25" | 8.375" x 8.5" (21.26 x 21.56 cm) |
| 8" x 10" | 8.125" x 10.25" (20.52 x 26 cm) |

## Implementation Constants

### For `types.ts` / Application Config

```typescript
// KDP-optimized page sizes for coloring books
export const KDP_PAGE_SIZES = [
  { 
    id: 'square', 
    label: '8.5" × 8.5"', 
    width: 8.5, 
    height: 8.5,
    aspectRatio: '1:1',
    mmWidth: 215.9,
    mmHeight: 215.9,
    minPages: 24,
    maxPages: 590,
  },
  { 
    id: 'portrait', 
    label: '8.5" × 11"', 
    width: 8.5, 
    height: 11,
    aspectRatio: '3:4',  // Actually 17:22, but 3:4 is close enough for AI
    mmWidth: 215.9,
    mmHeight: 279.4,
    minPages: 24,
    maxPages: 590,
  },
];

// Margin specifications (for books 24-150 pages, no bleed)
export const KDP_MARGINS = {
  outside: 0.25,      // inches - top, bottom, outer edge
  outsideMm: 6.35,    // mm
  gutter: 0.375,      // inches - inside/spine edge (24-150 pages)
  gutterMm: 9.53,     // mm
};

// Safe content area calculation
export const getKdpSafeArea = (width: number, height: number) => ({
  width: width - KDP_MARGINS.outside - KDP_MARGINS.gutter,
  height: height - (KDP_MARGINS.outside * 2),
  // Percentage of total area that is "safe"
  safePercent: Math.round(
    ((width - KDP_MARGINS.outside - KDP_MARGINS.gutter) * (height - KDP_MARGINS.outside * 2)) /
    (width * height) * 100
  ),
});
```

### For AI Prompt Guidance

```typescript
// Add to prompts for margin-safe composition
const KDP_COMPOSITION_GUIDANCE = `
COMPOSITION FOR PRINT:
- Keep main subjects within the center 85% of the canvas
- Leave breathing room at all edges (minimum 0.25" / 6mm)
- Extra margin on left edge of odd pages, right edge of even pages (gutter/binding area)
- Content near edges may be trimmed or hidden in binding
`;
```
